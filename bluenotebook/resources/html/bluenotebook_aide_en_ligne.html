Aide en ligne de BlueNotebook

Introduction

Bienvenue dans la documentation de BlueNotebook. Cet éditeur a été conçu pour vous offrir une expérience simple et puissante pour la prise de notes quotidienne au format Markdown.

Gestion du Journal

Le concept central de BlueNotebook est le journal. C'est un dossier sur votre ordinateur où toutes vos notes sont stockées.

  • Fichier du jour : Au lancement, BlueNotebook ouvre ou crée automatiquement un fichier pour la date du jour (ex: 20240729.md).
  • Sauvegarde : Utiliser Fichier -> Sauvegarder (Ctrl+S) enregistre vos modifications dans le fichier du jour. Si le fichier existe déjà, vous pouvez choisir de remplacer son contenu ou d'ajouter vos nouvelles notes à la fin.
  • Changer de journal : Le menu Fichier -> Ouvrir Journal vous permet de sélectionner un autre dossier comme répertoire de journal.

Menu Formater

Le menu Formater vous permet d'appliquer rapidement la syntaxe Markdown à votre texte sans avoir à la taper manuellement.

  • Titre : Applique des niveaux de titre (#, ##, etc.).
  • Style de texte : Met en gras (**texte**), italique (*texte*), barré (~~texte~~) ou surligné (==texte==).
  • Code : Formate du code en ligne ou en 
    bloc de code.
  • Listes : Crée des listes à puces, numérotées ou des listes de tâches.
  • Insérer : Ajoute des éléments comme des liens, images, tableaux, lignes horizontales ou citations.
  • RaZ : Supprime tout le formatage Markdown de la sélection.

Raccourcis Clavier Utiles

  • Ctrl+N : Nouveau fichier (non journal).
  • Ctrl+O : Ouvrir un fichier spécifique.
  • Ctrl+S : Sauvegarder dans le journal du jour.
  • Ctrl+B : Mettre en gras.
  • Ctrl+I : Insérer une image (ou mettre en italique si l'extension est désactivée).
  • F5 : Afficher/Masquer le panneau d'aperçu.
