📋 BlueNotebook Web Clipper

Capturez n'importe quelle page web et collez-la directement dans BlueNotebook en Markdown.

⬇ Installation

📋 BlueNotebook Clip

↑ Glissez ce bouton vers la barre de marque-pages de votre navigateur


  1. Affichez la barre de marque-pages (Ctrl+Shift+B sur Chrome/Firefox)
  2. Glissez le bouton bleu ci-dessus dans la barre
  3. Ou : clic droit sur la barre → Ajouter un marque-page → coller le code

▶ Utilisation

  1. Sur une page web, sélectionnez optionnellement du texte
  2. Cliquez sur 📋 BlueNotebook Clip dans la barre de marque-pages
  3. Un panneau s'ouvre dans le navigateur :
    • Modifiez le titre, l'auteur, les tags si besoin
    • L'aperçu Markdown se met à jour en temps réel
    • Vous pouvez aussi modifier directement le Markdown dans l'aperçu
  4. Cliquez Copier & Fermer
  5. Dans BlueNotebook, collez (Ctrl+V) à l'endroit souhaité

📄 Format de la note générée

# Titre de l'article > **Source :** [https://example.com/article](https://example.com/article) > **Auteur :** Prénom Nom > **Capturé le :** 2026-02-19 @@clipping @@monTag --- Contenu de l'article en Markdown...

Sélection partielle : si du texte est sélectionné dans la page avant de cliquer, seule la sélection est capturée. Sinon, Readability extrait automatiquement l'article principal.

⚙ Notes techniques

  • Dépendances chargées à la volée : Turndown (HTML→Markdown) et Readability
  • Aucune donnée envoyée à un serveur externe (tout est local)
  • Fonctionne sur Chrome, Firefox, Edge (navigateurs modernes)
  • Source lisible : bluenotebook_webclipper.js
  • Bookmarklet : bluenotebook_webclipper.min.js
